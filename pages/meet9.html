Ringkasan Materi Pertemuan 9

Pertemuan 9 Dunia Usaha

A. Tujuan Pembelajaran

Setelah mengikuti materi pada pertemuan ke-9 ini mahasiswa mampu memahami teknik pengolahan data yang berbasis komputer untuk dunia usaha.

B. Uraian Materi

1. Manfaat Komputer Dalam Bisnis

Komputer dalam bisnis telah berkembang dari sekadar alat hitung dan pencatat menjadi alat multifungsi yang mendukung berbagai aktivitas bisnis, termasuk hiburan, keperluan kantor, analisis, sumber informasi, dan kontrol. Dalam transaksi rutin, mesin kasir mencatat semua transaksi, memungkinkan manajer mengontrol kegiatan melalui jaringan internet. Komputer juga digunakan untuk administrasi keuangan, statistik, data stok, prediksi penjualan, dan perencanaan bisnis masa depan, membutuhkan data valid yang diinput ke dalam software komputer untuk menghasilkan laporan bagi pengambilan kebijakan.

2. Evolusi Pengolahan Data

Perdagangan yang semakin besar dan bebas memunculkan persaingan ketat, menuntut evolusi dalam pengolahan data untuk mencegah kecurangan. Tahap evolusi sesuai dengan Kurva S meliputi penghematan biaya, pengalihan SDM ke pekerjaan yang tidak bisa digantikan komputer, pengendalian pembelian dan penjualan, serta analisis data untuk prediksi penjualan. Evolusi ini mengubah peranan manusia, seperti agen penjualan, kasir, teller, operator telepon, dan tukang pos yang tergantikan teknologi. Profesi baru muncul seperti programmer, data analyst, dan database administrator.

3. Dasar-dasar Database

Database adalah sistem untuk menyimpan data dalam kapasitas besar secara efisien tanpa redudansi, menjadi bagian tak terpisahkan dari aplikasi web. Data bisa berupa angka, teks, gambar, hyperlink, dan suara. Database dibandingkan dengan spreadsheet, di mana spreadsheet adalah kalkulator besar dan database adalah sistem pengarsipan elektronik. Tahapan proses pengolahan data meliputi pengumpulan data, input, processing, output, distribusi, dan penyimpanan.

4. Sistem Manajemen Basisdata

Sistem Manajemen Basisdata (DBMS) adalah software untuk mengolah data dalam database, membentuk sistem basis data melalui interaksi antara DBMS dan database. Tujuan DBMS termasuk memelihara dan mengelola data konsisten, mengganti tempat penyimpanan fisik, mempercepat proses pengolahan data, dan menjaga keamanan data. Komponen DBMS meliputi file manager, database manager, query processor, DML precompiler, dan DDL compiler. Kekurangan DBMS meliputi ukuran besar, kompleksitas, biaya, dan kemungkinan kegagalan.

5. Data Menguntungkan Bisnis

Data penting untuk bisnis kecil karena meningkatkan efisiensi dan profit, memberikan catatan tentang pelanggan dan produk, serta memungkinkan analisis tren. Database membantu bisnis memahami pelanggan, mengelola penjualan, mengontrol stok, menanggapi perubahan, dan meningkatkan keamanan data.

6. Suatu Usaha Tidak Menggunakan Database

Usaha kecil mungkin mengabaikan penggunaan database karena dianggap mahal, mengganggu, memakan waktu, sulit dipelajari, atau rendah prioritas. Namun, database dapat diimplementasikan secara bertahap, meningkatkan efisiensi bisnis dan memberikan manfaat jangka panjang.

C. Soal Latihan/Tugas

  1. Berikan pendapat anda mengenai DBMS!
  2. Jelaskan peranan administrator dalam pengelolaan database!
  3. Berikan contoh data yang digunakan untuk database dalam sebuah usaha!
  4. Sebutkan kegunaan database dalam kehidupan sehari-hari!
  5. Seberapa pentingkah database dalam persaingan usaha? Jelaskan!
